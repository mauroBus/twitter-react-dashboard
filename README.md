# Twitter Dash made with React & Redux (Demo app)

## Install
```
$ npm install
```

## Run
```
$ npm start
```
  Navigate to: [http://localhost:3000](http://localhost:3000)

## Build
```
$ npm run build
```

## Deploy (gh pages)
```
$ npm run deploy
```

## Docs
```
$ npm run docs
```
